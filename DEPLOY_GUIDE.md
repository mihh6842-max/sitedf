# Инструкция по деплою P2P Exchange на Netlify

## Вариант 1: Быстрый деплой (перетаскивание)

### Шаг 1: Подготовка файлов
1. Убедись, что в папке есть файлы:
   - `index.html`
   - `app.js`
   - `styles.css`

### Шаг 2: Деплой на Netlify
1. Перейди на https://app.netlify.com/
2. Войди через GitHub/Email
3. Нажми "Add new site" → "Deploy manually"
4. Перетащи папку `Новая папка` в окно браузера
5. Дождись завершения деплоя (30-60 секунд)
6. Получишь ссылку типа `https://random-name-123.netlify.app`

### Шаг 3: Настройка Telegram Bot
1. Открой @BotFather в Telegram
2. Отправь `/newbot`
3. Введи название бота (например: "P2P Exchange Bot")
4. Введи username бота (например: "p2p_exchange_bot")
5. Получишь токен бота
6. Отправь `/newapp`
7. Выбери своего бота
8. Введи название приложения
9. Введи описание
10. Загрузи иконку (512x512 px)
11. Введи URL с Netlify (например: `https://your-site.netlify.app`)
12. Готово!

## Вариант 2: Деплой через GitHub (рекомендуется)

### Шаг 1: Создание репозитория
1. Перейди на https://github.com/
2. Нажми "New repository"
3. Введи название (например: "p2p-exchange")
4. Выбери "Public"
5. Нажми "Create repository"

### Шаг 2: Загрузка файлов
```bash
cd "C:\Users\admin\Desktop\Новая папка"
git init
git add .
git commit -m "Initial commit"
git branch -M main
git remote add origin https://github.com/ВАШ_USERNAME/p2p-exchange.git
git push -u origin main
```

### Шаг 3: Подключение к Netlify
1. Перейди на https://app.netlify.com/
2. Нажми "Add new site" → "Import an existing project"
3. Выбери "GitHub"
4. Авторизуй Netlify в GitHub
5. Выбери репозиторий "p2p-exchange"
6. Настройки оставь по умолчанию:
   - Build command: (пусто)
   - Publish directory: (пусто)
7. Нажми "Deploy site"
8. Дождись завершения деплоя

### Шаг 4: Настройка домена (опционально)
1. В настройках сайта на Netlify нажми "Domain settings"
2. Нажми "Add custom domain"
3. Введи свой домен или используй бесплатный поддомен Netlify

## Важно: Ограничения localStorage

⚠️ **Текущая версия использует localStorage** - данные видны только локально на одном устройстве.

Для синхронизации между пользователями нужен Firebase:

### Настройка Firebase (для реальной работы)

1. Перейди на https://console.firebase.google.com/
2. Нажми "Добавить проект"
3. Введи название: "p2p-exchange"
4. Отключи Google Analytics
5. Нажми "Создать проект"

6. В левом меню выбери "Realtime Database"
7. Нажми "Создать базу данных"
8. Выбери регион: "europe-west1"
9. Выбери "Начать в тестовом режиме"
10. Нажми "Включить"

11. Перейди в "Правила" и вставь:
```json
{
  "rules": {
    "offers": {
      ".read": true,
      ".write": true
    },
    "deals": {
      ".read": true,
      ".write": true
    }
  }
}
```
12. Нажми "Опубликовать"

13. Нажми на шестеренку → "Настройки проекта"
14. Прокрутите вниз до "Ваши приложения"
15. Нажми на иконку веб-приложения (</>)
16. Введи название: "P2P Exchange Web"
17. Нажми "Зарегистрировать приложение"
18. Скопируй конфигурацию

19. **Отправь мне конфигурацию, я добавлю её в код**

## Решение проблем

### Заявки не видны другим пользователям
- Это нормально для localStorage
- Нужен Firebase для синхронизации

### Ошибка при создании заявки
- Открой консоль браузера (F12)
- Скопируй ошибку и отправь мне

### Сайт не открывается
- Проверь, что все файлы загружены
- Проверь консоль Netlify на ошибки
